Base UI + Tailwind CSS Plugin Example

Checkbox

Select Item

Apple (selected)
Banana (highlighted)

Popover

Popover content (open state)

Installation

npm install @base-ui/tailwindcss

Configuration

// tailwind.config.js
module.exports = {
  plugins: [
    require('@base-ui/tailwindcss')
  ],
}

Usage

<Checkbox.Root
  className="
    ui-checked:bg-green-500
    ui-disabled:opacity-50
  "
/>
