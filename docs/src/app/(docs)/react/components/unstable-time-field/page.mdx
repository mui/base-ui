## Anatomy

Import the component and assemble its parts:

```jsx title="Anatomy"
import { TimeField } from '@base-ui/react/time-field';

<TimeField.Root>
  <TimeField.Section />
  <TimeField.SectionList>
    <TimeField.Section />
  </TimeField.SectionList>
  <TimeField.Clear />
</TimeField.Root>;
```

## Examples

### Custom format

Use the `format` prop to customize the time display format. The format string follows [date-fns format patterns](https://date-fns.org/docs/format).

import { DemoTimeFieldCustomFormat } from './demos/custom-format';

<DemoTimeFieldCustomFormat />

### Clear button

Wrap the sections in `<TimeField.SectionList>` to render the clear button next to them using `<TimeField.Clear>`. The `SectionList` groups all sections into a single element so that additional controls can be placed alongside them inside the field.

import { DemoTimeFieldClearButton } from './demos/clear-button';

<DemoTimeFieldClearButton />

### Validation

Use the `minDate` and `maxDate` props to constrain the allowed time range. Combine with `<Field.Error>` to display validation messages.

import { DemoTimeFieldValidation } from './demos/validation';

<DemoTimeFieldValidation />

### Localization

Use `<TemporalLocaleProvider>` to set the locale and `placeholderGetters` to customize placeholder text.

The `placeholderGetters` prop is a temporary solution. It will no longer be needed once a built-in localization system is implemented.

import { DemoTimeFieldLocalization } from './demos/localization';

<DemoTimeFieldLocalization />
