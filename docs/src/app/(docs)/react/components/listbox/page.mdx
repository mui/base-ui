## Usage guidelines

- **Always visible**: Unlike [Select](/react/components/select), the Listbox is always open and does not use a popup or trigger. Use it when all options should be visible at once.
- **Multi-select support**: Set `selectionMode="multiple"` or `"explicit-multiple"` to allow selecting more than one item.
- **Form controls must have an accessible name**: Prefer `<Listbox.Label>`, or provide an `aria-label` on `<Listbox.List>` when no visible label is rendered. See the [forms guide](/react/handbook/forms).

## Anatomy

Import the component and assemble its parts:

```jsx title="Anatomy"
import { Listbox } from '@base-ui/react/listbox';

<Listbox.Root>
  <Listbox.Label />
  <Listbox.List>
    <Listbox.Item>
      <Listbox.ItemIndicator />
      <Listbox.ItemText />
      <Listbox.ItemDragHandle />
    </Listbox.Item>
    <Listbox.Group>
      <Listbox.GroupLabel />
    </Listbox.Group>
    <Listbox.LoadingTrigger />
  </Listbox.List>
</Listbox.Root>;
```

## Examples

### Selection modes

The `selectionMode` prop controls how items are selected:

- `"single"` (default) — clicking an item selects it and deselects any other.
- `"multiple"` — every click toggles an item. Hold Shift and click to select a contiguous range.
- `"explicit-multiple"` — clicking an item replaces the selection. Hold Ctrl/Cmd and click to toggle, or Shift and click to select a range. Touch and pen interactions toggle without a modifier, matching `"multiple"` behavior.
- `"none"` — selection is disabled entirely.

### Lazy loading

Use `Listbox.LoadingTrigger` as a sentinel at the end of the list. When it scrolls into view, the `onLoadMore` callback fires.

### Drag and drop

Set `draggable` on items and provide an `onItemsReorder` callback on the root. Use `Listbox.ItemDragHandle` to restrict drag initiation to a handle. Items can also be reordered with the keyboard using Alt+Arrow keys.

Set `draggable="within-group"` to constrain reordering within the item's parent `Listbox.Group`. Items cannot be dragged across group boundaries.

In `multiple` or `explicit-multiple` mode, dragging a selected item moves all selected items together, preserving their relative order.

### Custom keyboard shortcuts

You can add custom keyboard shortcuts by handling `onKeyDown` on `Listbox.List` and updating the items array directly. This example uses `[` and `]` to move the focused item to the beginning or the end of the list.
