## Anatomy

Import the component and render a basic editor:

```jsx title="Anatomy"
import { Editor } from '@base-ui/react/editor';

export default function BasicEditor() {
  return (
    <Editor placeholder="Start typing..." />
  );
}
```

For more advanced layouts, you can compose the editor components:

```jsx title="Compositional Anatomy"
import { Editor } from '@base-ui/react/editor';

export default function AdvancedEditor() {
  return (
    <Editor.Provider>
      <Editor.Toolbar />
      <Editor.Content />
      <Editor.FloatingToolbar />
    </Editor.Provider>
  );
}
```

## Usage

- Provide a helpful `placeholder` to guide users when the editor is empty.
- Use the `value` and `onChange` props to control the content with a serialized editor state.
- For more advanced editing controls, compose with `Editor.Toolbar` and other components from the `Editor` namespace.

### AI Autocomplete

The Editor can be configured with an AI provider to offer autocomplete suggestions.
The following example demonstrates a mock AI provider that suggests text based on the last word typed.

<DemoEditorAICompletion />

To enable AI completions, provide an `ai` object to the `Editor`:

```jsx
<Editor
  ai={{
    getCompletion: async (text) => {
      // Call your AI API here
      return " suggestion";
    },
    debounceMs: 500,
  }}
/>
```
