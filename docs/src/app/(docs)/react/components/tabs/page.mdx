## Anatomy

Import the component and assemble its parts:

```jsx title="Anatomy"
import { Tabs } from '@base-ui/react/tabs';

<Tabs.Root>
  <Tabs.List>
    <Tabs.Tab />
    <Tabs.Indicator />
  </Tabs.List>
  <Tabs.Panel />
</Tabs.Root>;
```

## Examples

### Links

Use the `<Tabs.LinkTab>` part to render tabs as anchor elements.

When rendering a router link, control `<Tabs.Root>` from the current route.
The router is the source of truth for navigation, so deriving the selected value from the URL keeps the active tab synchronized with clicks, back/forward navigation, redirects, and route changes that happen outside the tab list.
