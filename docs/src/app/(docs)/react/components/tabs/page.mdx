## Anatomy

Import the component and assemble its parts:

```jsx title="Anatomy"
import { Tabs } from '@base-ui/react/tabs';

<Tabs.Root>
  <Tabs.List>
    <Tabs.Tab />
    <Tabs.Indicator />
  </Tabs.List>
  <Tabs.Panel />
</Tabs.Root>;
```

## Examples

### Links

Use the `<Tabs.LinkTab>` part to render tabs as anchor elements.

The selected tab updates only when activation stays within the current page.
Modified clicks (Cmd/Ctrl/Shift/Alt or non-primary buttons), `target="_blank"`, and `download` links keep their default browser behavior and do not change the selection.
To set `target` or `download` on the underlying anchor, pass them through the `render` prop (for example, `render={<a target="_blank" />}`).
