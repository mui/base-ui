## API reference

Import the component and use it as a single part:

```jsx title="Anatomy"
import { NumberField } from '@base-ui-components/react/number-field';

<NumberField.Root>
  <NumberField.Group>
    <NumberField.Decrement />
    <NumberField.Input />
    <NumberField.Increment />
    <NumberField.ScrubArea>
      <NumberField.ScrubAreaCursor />
    </NumberField.ScrubArea>
  </NumberField.Group>
</NumberField.Root>;
```
