## Usage guidelines

- **Do not wrap in a label**: A checkbox should not be wrapped in a `<label>` element, because it renders both a `<button>` and an `<input>`, which would result in invalid HTML. Instead, use a separate `<label>` element and link it to the checkbox using the `htmlFor` attribute.

## Anatomy

Import the component and assemble its parts:

```jsx title="Anatomy"
import { Checkbox } from '@base-ui-components/react/checkbox';

<Checkbox.Root>
  <Checkbox.Indicator />
</Checkbox.Root>;
```

## Examples

### Form integration

To use a single checkbox in a form, pass the checkbox's `name` to `<Field>` and use `<Field.Label>` to label the control.

```tsx title="Using Checkbox in a form"
<Form>
  <Field.Root name="stayLoggedIn">
    <Checkbox.Root />
    <Field.Label>Stay logged in for 7 days</Field.Label>
  </Field.Root>
</Form>
```
