## Anatomy

Import the component and assemble its parts:

```jsx title="Anatomy"
import { Select } from '@base-ui-components/react/select';

<Select.Root>
  <Select.Trigger>
    <Select.Value />
    <Select.Icon />
  </Select.Trigger>

  <Select.Portal>
    <Select.Backdrop />
    <Select.Positioner>
      <Select.ScrollUpArrow />
      <Select.Popup>
        <Select.Arrow />
        <Select.Item>
          <Select.ItemText />
          <Select.ItemIndicator />
        </Select.Item>
        <Select.Separator />
        <Select.Group>
          <Select.GroupLabel />
        </Select.Group>
      </Select.Popup>
      <Select.ScrollDownArrow />
    </Select.Positioner>
  </Select.Portal>
</Select.Root>;
```

## Formatting the value

By default, the `Select.Value` component will render the raw `value`.
To render a formatted value, a lookup map is needed to match the value to its `Select.ItemText` (`labels` below):

```jsx title="Lookup map"
const labels = new Map([
  [null, 'Select value'],
  [1, 'One'],
  [2, 'Two'],
  [3, 'Three'],
]);

<Select.Value>{(value) => labels.get(value)}</Select.Value>;
```

If you have a data structure, you can pass an `items` prop to `Select.Root`, which will be used to render the matching label for the `Select.Value` component instead.

```jsx title="Record data structure" {7}
const items = {
  null: 'Select value',
  1: 'One',
  ...
};

<Select.Root items={items}>
  <Select.Value />
</Select.Root>;
```

```jsx title="Array data structure" {7}
const items = [
  { value: null, label: 'Select value' },
  { value: 1, label: 'One' },
  ...
];

<Select.Root items={items}>
  <Select.Value />
</Select.Root>;
```
