## API reference

Import the component and place its parts the following way:

```jsx title="Anatomy"
import { ScrollArea } from '@base-ui-components/react/scroll-area';

<ScrollArea.Root>
  <ScrollArea.Viewport />
  <ScrollArea.Scrollbar>
    <ScrollArea.Thumb />
  </ScrollArea.Scrollbar>
  <ScrollArea.Corner />
</ScrollArea.Root>;
```
