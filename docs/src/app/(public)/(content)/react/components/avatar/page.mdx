## API reference

Import the component and assemble its parts:

```jsx title="Anatomy"
import { Avatar } from '@base-ui-components/react/avatar';

<Avatar.Root>
  <Avatar.Image src="" />
  <Avatar.Fallback>LT</Avatar.Fallback>
</Avatar.Root>;
```
