## API reference

Import the component and place its parts the following way:

```jsx title="Anatomy"
import { Progress } from '@base-ui-components/react/progress';

<Progress.Root>
  <Progress.Track>
    <Progress.Indicator />
  </Progress.Track>
</Progress.Root>;
```
