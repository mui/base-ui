## API reference

Form is composed together with [Field](/react/components/field). Import the components and place them together:

```jsx title="Anatomy"
import { Field } from '@base-ui-components/react/field';
import { Form } from '@base-ui-components/react/form';

<Form>
  <Field.Root>
    <Field.Label />
    <Field.Control />
    <Field.Error />
  </Field.Root>
</Form>;
```
