## Anatomy

Import the components and place them together:

```jsx title="Anatomy"
import { Combobox } from '@base-ui-components/react/combobox';

<Combobox.Root>
  <Combobox.Input />
  <Combobox.Value />
  <Combobox.List />
  <Combobox.Trigger />
  <Combobox.Portal>
    <Combobox.Positioner>
      <Combobox.Popup>
        <Combobox.Status />
        <Combobox.Item>
          <Combobox.ItemIndicator />
        </Combobox.Item>
        <Combobox.Separator />
        <Combobox.Group>
          <Combobox.GroupLabel />
        </Combobox.Group>
      </Combobox.Popup>
    </Combobox.Positioner>
  </Combobox.Portal>
</Combobox.Root>;
```

### Input autofill

The combobox's input can be autofilled with the highlighted item when navigating with arrow keys before selecting an item.

### Inside `Popover`

Using `Combobox.List` instead of `Combobox.Popup`, the combobox can be rendered inline inside a parent popover to create a searchable select menu by adding the `selectable` prop to `Combobox.Root`.
