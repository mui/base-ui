## Anatomy

Import the components and place them together:

```jsx title="Anatomy"
import { Combobox } from '@base-ui-components/react/combobox';

<Combobox.Root>
  <Combobox.Input />
  <Combobox.Value />
  <Combobox.Trigger />

  <Combobox.Portal>
    <Combobox.Positioner>
      <Combobox.Popup>
        <Combobox.Status />
        <Combobox.List>
          <Combobox.Item>
            <Combobox.ItemIndicator />
          </Combobox.Item>
          <Combobox.Separator />
          <Combobox.Group>
            <Combobox.GroupLabel />
          </Combobox.Group>
        </Combobox.List>
      </Combobox.Popup>
    </Combobox.Positioner>
  </Combobox.Portal>
</Combobox.Root>;
```

The combobox can be rendered inline inside a parent popover to create a searchable select menu by adding the `selectable` prop to `Combobox.Root`.
