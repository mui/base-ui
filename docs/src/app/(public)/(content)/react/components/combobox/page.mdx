## Anatomy

Import the components and place them together:

```jsx title="Anatomy"
import { Combobox } from '@base-ui-components/react/combobox';

<Combobox.Root>
  <Combobox.SelectedValue />
  <Combobox.Trigger />

  <Combobox.Chips>
    <Combobox.Chip>
      <Combobox.ChipRemove />
    </Combobox.Chip>

    <Combobox.Input />
    <Combobox.Clear />
  </Combobox.Chips>

  <Combobox.Portal>
    <Combobox.Positioner>
      <Combobox.Popup>
        <Combobox.Status />
        <Combobox.Empty />

        <Combobox.List>
          <Combobox.Row>
            <Combobox.Item>
              <Combobox.ItemIndicator />
            </Combobox.Item>
          </Combobox.Row>

          <Combobox.Separator />

          <Combobox.Group>
            <Combobox.GroupLabel />
          </Combobox.Group>

          <Combobox.Collection />
        </Combobox.List>
      </Combobox.Popup>
    </Combobox.Positioner>
  </Combobox.Portal>
</Combobox.Root>;
```

## Examples

### Select

The combobox can require the input value to match one of its predefined options. Trigger and clear buttons are also added to the input container.

<Demo path="./demos/option-only" compact />

### Multiple select

The combobox can be used to create a multiple select menu, rendering chips for each selected value in the input.

<Demo path="./demos/multiple" compact />

### Autocomplete

The combobox popup can only show while the query matches at least one item.

<Demo path="./demos/autocomplete" compact />

### Without autocomplete

The combobox can display predefined options without filtering or selection state, demonstrating search history.

<Demo path="./demos/without-autocomplete" compact />

### Async search

The combobox items can be loaded asynchronously.

<Demo path="./demos/async" compact />

### Input autofill

The combobox's input can be autofilled with the highlighted item when navigating with arrow keys before selecting an item.

<Demo path="./demos/autofill" compact />

### Grouped

The combobox can group related options together by passing an array of group objects to the `items` prop.

<Demo path="./demos/grouped" compact />

### Inside `Popover`

The combobox can be rendered inline inside a parent popover to create a searchable select menu.

<Demo path="./demos/inside-popover" compact />

### Grid layout

The combobox items can be navigated in a grid layout by wrapping each row in `Combobox.Row` adding the `cols` prop to `Combobox.Root`.

<Demo path="./demos/grid" compact />

### Command menu

The combobox can be used to create a command menu when rendered inside a `Dialog` component.

<Demo path="./demos/command-menu" compact />

### Fuzzy matching

The combobox can use fuzzy matching to find relevant results even when the search query doesn't exactly match the item text. This demo uses the `match-sorter` library to implement intelligent search capabilities.

<Demo path="./demos/fuzzy-matching" compact />

### Virtualized

The combobox can efficiently handle large datasets using a virtualization library like `@tanstack/react-virtual`. A combobox with 10,000 items maintains smooth performance by only rendering items within a visible window.

<Demo path="./demos/virtualized" compact />

### Add option

The combobox can allow users to add new options that are not in the predefined list.

<Demo path="./demos/add-option" compact />
