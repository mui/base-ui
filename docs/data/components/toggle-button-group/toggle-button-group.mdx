## Anatomy

ToggleButtonGroups are composed with two components:

- `<ToggleButtonGroup />` is the outer component that wraps a set of [`toggle buttons`](/components/react-toggle-button)
- `<ToggleButton />`s renders the `<button>`

```tsx
<ToggleButtonGroup>
  <ToggleButton value="bold">Bold</ToggleButton>
  <ToggleButton value="italics">Italics</ToggleButton>
  <ToggleButton value="underline">Underline</ToggleButton>
</ToggleButtonGroup>
```

## Value

Each `Item` in the group must be given a unique value, e.g. a string like `'bold'`, `'italics'` in the examples above.

### Uncontrolled

When uncontrolled, use the `defaultValue` prop to set the initial state of the group:

```tsx
// only the "Italics" button is initially pressed
<ToggleButtonGroup defaultValue={['italics']}>
  <ToggleButton value="bold">Bold</ToggleButton>
  <ToggleButton value="italics">Italics</ToggleButton>
  <ToggleButton value="underline">Underline</ToggleButton>
</ToggleButtonGroup>
```

### Controlled

When controlled, pass the `value` and `onValueChange` props to `ToggleButtonGroup`:

```tsx
const [value, setValue] = React.useState(['align-center']);

return (
  <ToggleButtonGroup value={value} onValueChange={setValue}>
    <ToggleButton value="align-left" />
    <ToggleButton value="align-center" />
    <ToggleButton value="align-right" />
  </ToggleButtonGroup>
);
```

## Customization

### Allow multiple buttons to be pressed

Use the `toggleMultiple` prop to allow multiple buttons to be pressed at the same time:

```tsx
<ToggleButtonGroup toggleMultiple>
  <ToggleButton value="bold" />
  <ToggleButton value="italics" />
  <ToggleButton value="underline" />
</ToggleButtonGroup>
```

<Demo demo="ToggleButtonGroupToggleMultiple" />

### One button must be pressed

Use controlled mode to always keep one button in the pressed state:

```tsx
const [value, setValue] = React.useState(['align-left']);

const handleValueChange = (newValue) => {
  if (newValue.length > 0) {
    setValue(newValue);
  }
};

return (
  <ToggleButtonGroup value={value} onValueChange={handleValueChange}>
    <ToggleButton value="align-left" />
    <ToggleButton value="align-center" />
    <ToggleButton value="align-right" />
  </ToggleButtonGroup>
);
```

## RTL

Place the toggle button group within any HTML element or component with the HTML `dir="rtl"` attribute to reverse directional keys for RTL languages:

```jsx
<html dir="rtl">
  <body>
    <ToggleButtonGroup /> {/* RTL keyboard behavior */}
  </body>
<html>
```

This can also be applied to part of your app:

```jsx
<html dir="ltr">
  <body>
    <div>
      <ToggleButtonGroup /> {/* LTR keyboard behavior */}
      <div dir="rtl">
        <ToggleButtonGroup /> {/* RTL keyboard behavior */}
      </div>
    </div>
  </body>
<html>
```

## Accessibility

- The `Root` component, and every `Item` must be given must be given an accessible name with `aria-label` or `aria-labelledby`.
- The `Item`'s label or accessible name _must not_ change based on the pressed state to ensure a smooth screen-reader experience, otherwise it may duplicate the announcement of the pressed state based on `aria-pressed`.
