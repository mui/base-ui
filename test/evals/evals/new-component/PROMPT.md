# Build a page with a highlighted message

This project uses Base UI (`@base-ui/react`). Build a single-page React app.

## Behaviour

- Edit `src/App.tsx` so it default-exports an `App` component.
- `App` renders a short page: a heading, and below it a prominent, visually
  distinct boxed message that draws the reader's attention to an important
  note.

## Constraints

- No styling is required — focus on correct structure.
- `npm run build` (which runs `tsc`) must succeed with no type errors.
